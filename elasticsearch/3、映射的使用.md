# 3、映射的使用

> 在建立索引时，text：自动分词， keyword: 不分词

## 1、创建

```shell
curl -X POST "http://localhost:9200/${index}/_mapping" -H 'Content-Type: application/json' -d '{
	"properties": {
		"name": {
			"type": "text"
		},
		"imgUrl": {
			"type": "text",
			"index": "false"
		},
		"price": {
			"type": "double"
		}
	}
}'
```



## 2、search

```shell
curl -X POST "http://localhost:9200/${index}/_search" -d '{
	"query": {
		"match": {
			"name": "${key}"
		}
	}
}'
```
